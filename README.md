## Quick Start

### Local Development

```bash
cd ai-tools-report
python -m http.server 8000
```

Then visit: http://localhost:8000

### Deploy to Vercel

1. Push to GitHub:
```bash
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/ai-tools-report.git
git push -u origin main
```

2. Go to https://vercel.com
3. Click "New Project"
4. Import your GitHub repo
5. Click "Deploy" (no build config needed)

## Monetization Setup

### Google AdSense

1. Sign up at https://adsense.google.com
2. Add your site to AdSense
3. Get your publisher ID
4. Replace `ca-pub-XXXXXXXXXXXXXXXX` in `index.html` line with your ID
